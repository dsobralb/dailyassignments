# Requirements
- Must use at least one for loop and a while loop.
- Must ask user for a letter OR the whole word to be guessed
- Display  underscores for unknown letters and the correct letters in the correct places as they are guessed
- Display incorrect letter attempts in its own space.
- Display how many more attempts the user has before they lose the game.
- When a user guesses all correct letters OR guesses the whole word correctly display an alert that the user won and end the game.
- When a user runs out of guesses alert the user that they lost and what the correct word was. End the game.

There are no tests for this assignment
